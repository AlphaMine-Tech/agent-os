# Handoffs

Use this directory for concise active handoff notes when a task is in-flight and another agent may need to take over quickly.

Do not use handoffs as a substitute for:
- TODO.md
- MEMORY.md
- project docs
- decision records
- build-state files

Handoffs should be short, current, and disposable once their contents are absorbed into the canonical files.

See `docs/workflows/handoff-protocol.md` for the full handoff format and protocol.
