# Agent-OS Index

This directory holds helper maps for agents to quickly understand your infrastructure:

- `hosts.md` — server/machine inventory with SSH details
- `repos.md` — git repositories and their locations
- `services.md` — running services and their ports
- `domains.md` — domain names and DNS routing
- `network-topology.md` — network layout and routing

Populate these as needed when they provide durable value. Keep them current — stale infrastructure docs are worse than no docs.
